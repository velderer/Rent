# Техническое задание (ТЗ)

Проект

Разработка современного сайта-визитки для сдачи в аренду 5 квартир.

Основная задача сайта — максимально просто и быстро предоставить пользователю информацию о квартирах и мотивировать оставить заявку на аренду.

⸻

1. Общая информация

Тип сайта:

	•	Сайт-визитка
	•	Каталог из 5 квартир
	•	Адаптивный дизайн
	•	Административная панель

Языки:

	•	Первый этап — русский.
	•	Архитектура должна позволять легко добавить другие языки в будущем.

⸻

2. Цели сайта

Пользователь должен за минимальное количество действий:

	•	увидеть все квартиры;
	•	выбрать понравившуюся;
	•	посмотреть фотографии;
	•	ознакомиться с характеристиками;
	•	узнать стоимость;
	•	проверить доступность;
	•	оставить заявку.

⸻

3. Стиль сайта

Дизайн должен выглядеть дорого, современно и минималистично.

Ассоциации:

	•	Premium
	•	Modern
	•	Clean
	•	Luxury
	•	User Friendly
	•	Fast
	•	Fresh

Цветовая палитра

Основные цвета:

	•	чёрный
	•	белый
	•	яркий фосфорный (неоновый лайм/зелёный)

Использование:

Фон:

	•	белый либо чёрный (в зависимости от блока)

Акцент:

	•	кнопки
	•	ссылки
	•	переключатели
	•	активные элементы
	•	иконки
	•	hover-эффекты

Не использовать яркие градиенты, кислотный дизайн и перегруженность.

Минимализм.

8. Адаптивность

Полностью адаптивный сайт.

Поддержка:

	•	ПК
	•	ноутбуков
	•	планшетов
	•	смартфонов

Особое внимание уделить мобильной версии, так как большинство пользователей будут заходить со смартфонов.

⸻

9. Скорость

Сайт должен открываться максимально быстро.

Использовать:

	•	оптимизацию изображений;
	•	lazy loading;
	•	современные форматы изображений (WebP/AVIF);
	•	кэширование.

⸻

10. SEO

Каждая квартира имеет:

	•	собственный URL;

	•	Open Graph для социальных сетей.

/apartments/apartment-1

13. Безопасность

	•	Авторизация администратора.
	•	Защита форм от спама (Google reCAPTCHA или аналог).
	•	Валидация всех данных.
	•	Защита панели администратора.
	•	Безопасная загрузка изображений.

⸻

14. Технологические требования

	•	Современный стек разработки.
	•	Быстрая работа интерфейса.
	•	Адаптивная верстка.
	•	Чистый и поддерживаемый код.
	•	Возможность дальнейшего масштабирования (добавление новых квартир, языков и функций без полной переработки проекта).

⸻

15. Итоговый результат

В результате разработки должен быть получен современный премиальный сайт-визитка с каталогом из 5 квартир, ориентированный на высокую конверсию. Пользователь за несколько секунд сможет выбрать подходящую квартиру, изучить фотографии и характеристики, посмотреть расположение на карте и оставить заявку. Администратор сможет полностью управлять контентом через удобную панель: изменять фотографии, описание, стоимость, характеристики, удобства, статус доступности и просматривать все поступающие заявки без участия разработчика. Сайт должен быть быстрым, интуитивно понятным, визуально привлекательным, адаптированным под все устройства и готовым к дальнейшему развитию.Rent
